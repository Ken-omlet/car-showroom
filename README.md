# 🚗 Car Showroom

A project car showroom website built with **Next.js**.  
It features a cinematic UI, car gallery, search, animations, and dark/light mode.

---

## 🌐 Live Demo

https://ken-car-showroom.vercel.app

---

## ✨ Features

### 🚘 Car Gallery
- Display luxury cars with images
- Show price and details
- Animated cards

### 🔍 Search
- Search cars by name instantly
- Real-time filtering

### 🌙 Theme
- Dark / Light mode toggle
- Smooth UI transitions

### 🎥 UI Design
- Full-screen video hero section
- Smooth scrolling navigation
- Modern luxury design

---

## 🛠️ Built With

- Next.js – Frontend Framework  
- React – UI Library  
- Tailwind CSS – Styling  
- Framer Motion – Animations  
- Vercel – Deployment
